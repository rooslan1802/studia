# Studia

Desktop-приложение (Electron + React + SQLite) для управления студиями, детьми, посещениями и оплатами.

## Запуск
```bash
npm install
npm run db:reset   # опционально: сбросить локальную БД и заново загрузить seed
npm start
```

## Сборка macOS
```bash
npm run build:mac
```

## Что есть сейчас
- `Dashboard` с кликабельными карточками.
- `Структура`: города карточками, внутри кружки и группы, у групп отдельное расписание.
- `Дети`: список по всем городам + фильтры, компактные иконки действий.
- `Посещения`: табель-матрица в стиле excel, даты берутся из расписания группы.
- `Оплаты`: комментарии, обещанная дата, статус в срок/с опозданием, CSV экспорт.
- `Уведомления`.

## Бизнес-правила
- Платник платит за каждые 8 проведенных занятий.
- Пропуск ребенком занятие не переносит оплату (урок считается проведенным).
- Если занятие отменено студией, оно не засчитывается в оплату.
- При `lessonsCount >= 8` ребенок попадает в требующие оплаты.

## Seed для теста
- Город: `Астана`.
- Кружки: `Живопись`, `Футбол`, `Танцы`, `Шахматы`.
- У каждого кружка есть группы с разным расписанием.
- По 10 детей в каждом кружке.
